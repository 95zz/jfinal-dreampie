jfinal好用的插件正在开发，

下载源码:https://github.com/Dreampie/jfinal-dreampie,
maven引入方式：http://search.maven.org/ 输入jfinal-dreampie搜索:
```xml
<dependencies>
   <dependency>
      <groupId>cn.dreampie</groupId>
      <artifactId>jfinal-dreampie</artifactId>
      <version>1.2</version>
    </dependency>
</dependencies>
```
